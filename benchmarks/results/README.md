# Benchmark Results

This directory contains the output reports from running the COON LLM benchmarks.

## Files

- `comprehension.md` - Results from comprehension benchmark (understanding COON code)
- `generation.md` - Results from generation benchmark (creating COON code)

## Running Benchmarks

```bash
# From benchmarks directory
npm run benchmark:comprehension
npm run benchmark:generation
npm run benchmark:all
```

Reports are automatically saved here after each benchmark run.
